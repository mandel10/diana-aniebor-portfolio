# B&J Biscuit — Retail Sales Dashboard

**Overview**  
Interactive Excel dashboard showing sales performance for B&J Biscuit: monthly trends, brand performance, top merchants, payment method distribution, and geographic breakdown.

**Files**
- `B&J_Biscuit_Dashboard.xlsx` — Final interactive dashboard (Excel)
- `bj_biscuit_clean_data.xlsx` — Cleaned source data (table `SalesData`)
- `screenshots/dashboard_main.png` — Dashboard snapshot for preview

**How it was built**
- Data cleaning: removed empty rows, normalized dates, converted currency fields to numeric
- Calculations: Year, Month, Quarter columns; Profit calculated as `Net = Amount - Fee`
- Analysis: PivotTables for monthly revenue, category performance, payment-method split, top merchants
- Visuals: PivotCharts and Slicers for Year, Category, Country and Payment Method
- Tool: Microsoft Excel (Tables, PivotTables, PivotCharts, Slicers)
